All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/), and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

- Standard badges to README.md (Version, Status, Live Demo, License, Semantic Versioning).

- Required HTML comment header to `index.html` (Author, Description, App Name, Year).

- Link to companion tool: PDF Reorder.

## [1.1.0] - 2025-12-19

### Added
- GitHub Pages deployment automation via GitHub Actions.
- Live demo link in README.md.

## [1.0.0] - 2025-12-19

### Added
- Initial release of SplitPDF.
- Modern dark-mode UI with glassmorphism effects.
- Landscape-to-portrait PDF splitting logic using `pdf-lib`.
- Client-side processing for enhanced privacy.
- Drag-and-drop file upload functionality.
- Progress tracking and success feedback.
- Documentation: README.md, CHANGELOG.md, LICENSE, and .gitignore.
